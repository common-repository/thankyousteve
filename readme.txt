== Description ==

This plugin enables you to use the ThankYouSteve widget. Drag the widget into your sidebar and a box "Steve Jobs 1955 - 2011" with an external link to http://www.apple.com/stevejobs/ appears.

== Installation ==

1. Place the plugin in your wordpress plugin directory.
2. Active the Plugin in Wordpress.
3. Drag the widget into your sidebar.
